Script written in Python

Should work on any machine running Python 2.7+

Clone Github repo from:
https://github.com/mHart1000/Subset-Sum.git

From windows or linux command line, enter the path to the python script followed by "<" followed by the path to the data file
Example in Windows:
c:\users\matthew\desktop\subset-sum\app.py < c:\users\matthew\desktop\data.csv
